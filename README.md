# 🌑 InStudy / disto.mveu.ru — Mono UI

Tampermonkey-скрипт — полный UI-оверхаул студенческого портала [disto.mveu.ru](https://disto.mveu.ru) (InStudy, МВЕУ): монохромная палитра, 5 акцентных цветовых схем, переработанная вёрстка, оптимизация производительности.

![screenshot](screenshot.png)

## ⚙️ Установка

1. Установите расширение [Tampermonkey](https://www.tampermonkey.net/) для вашего браузера.
2. Откройте **Tampermonkey → Настройки → Утилиты → Импорт из URL** и вставьте ссылку:
   ```
   https://raw.githubusercontent.com/juushimatsu/instudy-mveu-ui-fix-tampermonkey/main/main.js
   ```

## ✨ Возможности

- **Тёмная + светлая тема** — переключение кнопкой в шапке, выбор сохраняется в `localStorage`
- **5 акцентных цветовых схем** — Mono (нейтральный), Ocean (синий), Crimson (красный), Amber (оранжевый), Forest (зелёный). При выборе цветного акцента вся страница окрашивается в тёмные (dark) или пастельные (light) оттенки этого цвета; переключение темы автоматически пересчитывает фоны
- **Режим плотности** — кнопка в шапке переключает `compact → normal → wide` (размер шрифта, отступы, высота шапки)
- **Переработанное боковое меню** — узкая рейка 68 px, раскрывается по hover до 280 px с анимацией
- **Шрифты** — JetBrains Mono (UI / моноширинный) + Unbounded (заголовки)
- **Фиксы вёрстки** — шапка (`#status_bar`), форма входа, чат (стиль мессенджера, разделение «свои / чужие»), CRUD-таблицы, бейджи, аватары
- **Lazy-load списка преподавателей** — `.gulist` подгружается порциями по 60 при скролле + `content-visibility: auto`
- **Авто-фолбек аватаров** — сломанные `<img>` заменяются на инициал в круге
- **Отключение цветовых тем портала** — `blue.css` / `pink.css` деактивируются, inline-стили белого фона перебиваются
- **MutationObserver** — все правки применяются и к AJAX-контенту

- **Мигающие уведомления о новых сообщениях** — когда вкладка неактивна, заголовок мигнёт "📩 Новое сообщение"
- **Превью изображений прямо в чате** — файлы `.jpg/.png/.gif/.webp`, отправленные в сообщениях, автоматически распаковываются из zip и показываются как превью с лайтбоксом по клику
- **Атмосферные эффекты** — кнопка в шапке переключает между дождём, снегом и выключенным состоянием (частицы на canvas, без нагрузки на UI)
